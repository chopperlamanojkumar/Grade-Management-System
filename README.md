# Grade Management System

## Overview

The Grade Management System is a Java console application developed to manage student marks efficiently. It allows users to add student records, calculate averages, assign grades, search for students, generate reports, and analyze performance using arrays and object-oriented programming concepts.

## Features

- Add Student Marks
- View Student Records
- Search Student by Name
- Calculate Average Marks
- Find Highest and Lowest Marks
- Generate Performance Report
- Grade Calculation (A, B, C, D, F)
- Input Validation
- Menu Driven Application
- Object-Oriented Design

## Technologies Used

- Java
- VS Code
- JDK
- Arrays
- Object-Oriented Programming

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Navigate to the `src` folder.

Compile:

```bash
javac *.java
```

Run:

```bash
java GradeManagementSystem
```
